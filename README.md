## Cerberus simulator
This GitHub repository is part of the artifacts of the paper published at NDSS'26 [[DOI]](https://dx.doi.org/10.14722/ndss.2026.241857):
> On the Security Risks of Memory Adaptation and Augmentation in Data-plane DoS Mitigation

This repository provides the Cerberus [[SP'24]](https://ieeexplore.ieee.org/document/10646662) simulator for testing the Heracles attack.   
Our main artifact repository for the Heracles attack, Cerberus++, and Shield is available at [here](https://github.com/NetSP-KAIST/shield).
